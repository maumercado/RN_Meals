# RN Meals

React Native application for browsing and viewing meal recipes by category.

## Features

- Browse meals by category
- View detailed meal recipes
- Favorite meals functionality
- Recipe instructions and ingredients
- Meal images and descriptions

## Requirements

- Node.js 18 or above
- React Native development environment

## Installation

1. Clone this repository
2. Run `npm install`
3. For iOS: `npm run ios`
4. For Android: `npm run android`

## Technologies

- React Native
- React Navigation
- Context API for state management
